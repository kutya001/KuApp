# UI/UX Спецификация — KuApp Platform

**Версия:** 1.0  
**Дата:** 06.04.2026  
**Платформа:** PWA (Progressive Web Application)  
**Язык интерфейса:** Русский

---

## 1. Общие принципы дизайна

### 1.1 Визуальный стиль

| Параметр | Значение |
|----------|----------|
| Тема | Тёмная (Dark Mode) |
| Основной фон | `#0a0a0f` |
| Акцентный цвет | Неоновые оттенки (зависит от приложения) |
| Шрифты | Manrope (основной), JetBrains Mono (код/моноширинный) |
| Иконки | Emoji-иконки для приложений |
| Стиль | Минимализм, неоновые акценты, «arcade» эстетика |

### 1.2 Адаптивность

| Breakpoint | Устройство | Навигация |
|------------|-----------|-----------|
| `< 768px` | Мобильные устройства | Bottom navigation bar (5 табов) |
| `≥ 768px` | Планшеты и десктоп | Left sidebar |

### 1.3 PWA поведение

| Параметр | Значение |
|----------|----------|
| Display mode | `standalone` (без адресной строки браузера) |
| Ориентация | Portrait (портретная) |
| Установка | «Add to Home Screen» через manifest.json |
| Оффлайн | App shell кэшируется, данные требуют сети |

- Back button браузера также работает

## 3. Компоненты UI

### 3.1 Toast-уведомления

```
┌─────────────────────────────────────┐
│                                     │
│  ┌───────────────────────────────┐ │
│  │ ✅ Результат сохранён!        │ │  ← Success (зелёный)
│  └───────────────────────────────┘ │
│                                     │
│  ┌───────────────────────────────┐ │
│  │ ❌ Ошибка подключения         │ │  ← Error (красный)
│  └───────────────────────────────┘ │
│                                     │
│  ┌───────────────────────────────┐ │
│  │ ℹ️ Загрузка...                │ │  ← Info (синий)
│  └───────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

**Параметры:**
- Появляются сверху экрана
- Авто-скрытие через 3 секунды (success/info)
- Error — требует ручного закрытия
- Анимация: slide-in сверху, fade-out

### 3.2 Индикатор загрузки (Loader)

```
┌─────────────────────────────────────┐
│                                     │
│           ┌───────┐                │
│           │  ⏳   │                │
│           │ Загрузка...            │
│           └───────┘                │
│                                     │
└─────────────────────────────────────┘
```

**Параметры:**
- Полупрозрачный overlay на весь экран
- Спиннер + текст
- Блокирует взаимодействие с UI

### 3.3 Кнопки

| Тип | Стиль | Пример |
|-----|-------|--------|
