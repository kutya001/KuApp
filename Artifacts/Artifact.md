# 📁 Artifact — Документация KuApp Platform

Полный пакет технической документации для проекта **KuApp** — мульти-приложение PWA на Google Sheets.

> 📖 Основную документацию проекта (описание приложений, настройка, деплой) см. в [`README.md`](../README.md) корня репозитория.

---

## 📋 Содержимое

### 🔄 Основные документы (параллельная разработка)

| Документ | Файл | Описание |
|----------|------|----------|
| **Техническая спецификация** | [`TECH_SPEC.md`](./TECH_SPEC.md) | Архитектура, компоненты, API, стек технологий, как добавить приложение |
| **API Contract (OpenAPI)** | [`API_CONTRACT.md`](./API_CONTRACT.md) | Эндпоинты GAS API, схемы данных, примеры запросов/ответов |
| **UI/UX Спецификация** | [`UI_UX_SPEC.md`](./UI_UX_SPEC.md) | Экраны, компоненты, цветовая палитра, типографика, анимации, accessibility |
| **Модель данных (ER-диаграмма)** | [`ER_DIAGRAM.md`](./ER_DIAGRAM.md) | Таблицы Google Sheets, связи, бизнес-правила, ограничения |

### 📋 Вытекающие документы

| Документ | Файл | Описание |
|----------|------|----------|
| **Критерии приёмки** | [`ACCEPTANCE_CRITERIA.md`](./ACCEPTANCE_CRITERIA.md) | 39 критериев в формате Gherkin (Given/When/Then) |
| **План тестирования** | [`TEST_PLAN.md`](./TEST_PLAN.md) | 75 тест-кейсов, матрица покрытия, отчётность |
| **Architecture Decision Records** | [`ADR.md`](./ADR.md) | 10 задокументированных архитектурных решений |
